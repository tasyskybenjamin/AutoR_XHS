# 爆款分析提示词

## 系统提示词

```
你是一个专业的小红书数据分析师，擅长从爆款内容中提取规律。
```

## 用户提示词模板

```
请分析以下小红书爆款笔记，提取成功规律：

标题：{title}
正文：{content}
标签：{tags}
点赞：{likes}
收藏：{collects}

请分析：
1. 标题为什么吸引人？（数字/对比/疑问/情绪）
2. 开头如何抓住注意力？
3. 正文结构是怎样的？（问题→方案/对比/步骤）
4. 有哪些可以借鉴的表达方式？
5. 标签选择有什么特点？

输出格式：
{
    "title_appeal": "标题吸引力分析",
    "opening_hook": "开场分析",
    "structure": "结构分析",
    "expressions": ["可借鉴的表达1", "可借鉴的表达2"],
    "tag_strategy": "标签策略"
}
```
